# Gerenciador-De-Vagas-CLI-

> Gerenciador de Vagas é um projeto focado em organização e produtividade, com o objetivo principal de ajudar você a acompanhar suas candidaturas a vagas de emprego.

Com ele, você pode cadastrar todas as aplicações que realizou, registrar detalhes importantes e acompanhar o andamento de cada processo seletivo. Isso permite uma visão mais clara do seu progresso, facilitando a análise do seu desempenho e ajudando a tomar decisões mais estratégicas na sua jornada profissional.

## 📦 Tecnologias e Ferramentas

- [Node.js]
- JavaScript ES6+
- Módulos com `import` e `export`
- Biblioteca `readline` (nativa do Node.js)
